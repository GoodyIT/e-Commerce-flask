# Inventory Run Guide

1. Create a Virtual Env
2. pip install the following dependencies into the virtual env:
    2a: pymongo
    2b: flask
    2c: flask_login
    2d: flask_wtf
    2e: werkzeug
    2f: jwt
    example: pip install flask
3. run mongo database in shell. mongod
4. In new shell window, flask run
